# Smart Fitness Trainer

A Node.js fitness tracking application with MongoDB.

## Setup Instructions

1. Clone the repository
```bash
git clone https://github.com/KKaranjaa/smart-fitness-trainer.git
cd smart-fitness-trainer
```

2. Install dependencies
```bash
npm install
```

3. Set up environment variables
- Create a `.env` file (copy from `.env.example`)
- Add your MongoDB URI and other settings

4. Start the server
```bash
npm start
```
